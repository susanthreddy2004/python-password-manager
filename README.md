# 🔐 Python Password Manager

A secure Password Manager built with Python that helps users safely store and manage their passwords using encryption.

## Features

- Master Password Authentication
- Secure Password Storage
- Password Encryption & Decryption
- Random Password Generator
- SQLite Database Support
- Auto-Lock Security Feature
- User-Friendly Interface

## Technologies Used

- Python
- Tkinter
- SQLite
- Cryptography (Fernet)
- bcrypt

## How It Works

- Users create a master password to access the application.
- Passwords are encrypted before being stored.
- Stored credentials can be viewed, updated, and deleted securely.
- Strong passwords can be generated automatically.

## Security Features

- Master password hashing using bcrypt.
- Password encryption using Fernet.
- Secure storage of credentials.
- Auto-lock protection during inactivity.

## Project Goal

The goal of this project is to provide a simple and secure solution for managing passwords while demonstrating Python programming, encryption techniques, database management, and GUI development.
